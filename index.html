MINICRAFT 3D

Modo Criativo • Voxel 3D com Raycasting • Feito com HTML + CSS + JS + Three.js

Mundo 3D • Dia • Criativo
COMO JOGAR (3D):
WASD = mover
Mouse = olhar ao redor
ESPAÇO / Shift = subir / descer
Clique ESQUERDO = quebrar bloco
Clique DIREITO = colocar bloco
1~5 = trocar bloco
ESC = soltar mouse
R = regenerar mundo
Raycasting preciso + destaque do bloco!
